# Code of Conduct

## Our Standards

We are committed to a respectful, harassment-free community.

Examples of positive behavior:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the project

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory or hateful language
- Deliberate disruption of discussions

## Scope

This code of conduct applies in project spaces such as issues, pull requests, and discussions.

## Enforcement

Maintainers may remove, edit, or reject contributions and comments that violate this code of conduct.

## Reporting

Report violations privately to project maintainers.
