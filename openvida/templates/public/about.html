About OpenVIDA

OpenVIDA exists to help extend the life of Volvo vehicles by making parts and service information easier to access and use.

The core goal is simple: keep more Volvo vehicles maintainable, repairable, and on the road longer.

Project Status & Support

This project is open source. View the code on GitHub, report bugs or request features in the issue tracker, and contribute by opening an issue or submitting a pull request.

The project is community-driven and actively open to improvements. For bug reports and feature requests, use the issue tracker. To contribute, open an issue to discuss changes or submit a pull request.
